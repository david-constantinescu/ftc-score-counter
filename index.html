Capabilities

Everything you need,
nothing you don't.

Purpose-built for FTC competition scoring with real-time computer vision.

📹

Dual Camera Streams

Simultaneous Blue & Red goal monitoring via MJPEG. Supports USB webcams and IP cameras like DroidCam.

🧠

YOLOv8 + HSV Detection

Hybrid detection pipeline combining neural network object detection with colour-based HSV filtering for maximum reliability.

⚡

Real-time Scoring

SSE-powered live score updates in your browser. Ball counts update the instant they're detected — no page refresh needed.

📱

IP Camera Support

Connect DroidCam or any MJPEG IP camera by address. Auto-detection tries your default DroidCam on startup.

⏱️

Match Timer

2:30 countdown with FTC audio. Auto-resets ball trackers and scores when a new match starts.

🚀

GPU Accelerated

Automatically detects and uses CUDA, Apple Silicon MPS, or CPU. Optimised threading for Intel machines.

Get Started

One command.
That's it.

Uses only tools already on your OS to install everything. Re-run to update.

macOS
# Works everywhere. Installs everything. 🤖 $ curl -fsSL https://raw.githubusercontent.com/david-constantinescu/ftc-score-counter/main/macos.sh | bash
# One-liner for any distro (apt, dnf, pacman) $ curl -fsSL https://raw.githubusercontent.com/david-constantinescu/ftc-score-counter/main/linux.sh | bash
# Installs as a systemd service — starts at boot $ curl -fsSL https://raw.githubusercontent.com/david-constantinescu/ftc-score-counter/main/linux-service.sh | bash # Manage the service $ sudo systemctl status ftc-scorer $ sudo journalctl -u ftc-scorer -f
# Run in PowerShell as Administrator PS> iwr -useb https://raw.githubusercontent.com/david-constantinescu/ftc-score-counter/main/windows.ps1 | iex
# Clone, create venv, run $ git clone https://github.com/david-constantinescu/ftc-score-counter.git $ cd ftc-score-counter $ python3 -m venv .venv && source .venv/bin/activate $ pip install -r requirements.txt $ python app.py

Then open http://localhost:2016 in your browser.

Workflow

Set up in minutes.

Three steps from zero to live scoring.

1

Install & Launch

Run the one-liner for your OS. It installs Git, Python, dependencies, and starts the server in the background.

2

Connect Cameras

Select USB webcams from the dropdown or click "Add IP Camera" for DroidCam / network cameras. Two cameras — one per goal.

3

Start the Match

Hit Start. The FTC timer audio plays, ball detection begins, and live scores update in your browser in real time.
